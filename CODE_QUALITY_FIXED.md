## 说明

### 为什么删除miniprogram目录？

`miniprogram/` 目录是微信云开发模板文件，包含：
- 云开发示例代码
- 大尺寸的示例图片（超过200K）
- 云函数相关文件

这些文件不是我们项目的代码，项目使用的是根目录下的 `pages/` 目录。

### 项目结构

正确的项目结构：
```
wechat-miniware-example/
├── pages/          # 我们的页面代码（根目录）
├── utils/          # 工具类
├── app.js          # 应用入口（根目录）
├── app.json        # 应用配置（根目录）
└── ...
```

不是：
```
wechat-miniware-example/
└── miniprogram/    # 这是云开发模板，已删除
```
