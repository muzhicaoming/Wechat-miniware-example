## 使用方法

### 手动测试

1. 打开 `qa-test-plan.md`
2. 按照测试用例逐一测试
3. 记录测试结果
4. 填写测试报告

### 自动化测试

1. 在微信开发者工具中打开项目
2. 打开"调试器" -> "Console"
3. 复制 `automated-test.js` 的内容到控制台
4. 执行测试：
   ```javascript
   tester.runAllTests()
   ```

## 注意事项

1. **测试环境**: 确保使用正确的测试环境
2. **数据清理**: 测试前清理本地存储数据
3. **完整测试**: 按照测试计划完整执行
