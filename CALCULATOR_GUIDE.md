# 计算器页面使用说明

## 当前布局

```
┌─────────┬─────────┬─────────┬─────────┐
│    C    │    ÷    │    ×    │    -    │
├─────────┼─────────┼─────────┼─────────┤
│    7    │    8    │    9    │    ×    │
├─────────┼─────────┼─────────┼─────────┤
│    4    │    5    │    6    │    -    │
├─────────┼─────────┼─────────┤         │
│    1    │    2    │    3    │    =    │
├─────────┴─────────┼─────────┤         │
│         0         │    .    │    +    │
└───────────────────┴─────────┴─────────┘
```

## 按钮说明

### 第一行（运算符行）
- **C**：清除（红色）
- **÷**：除法（蓝色）
- **×**：乘法（蓝色）
- **-**：减法（蓝色）

### 第二行
- **7 8 9**：数字按钮（浅灰色）
- **×**：乘法（蓝色）

### 第三行
- **4 5 6**：数字按钮（浅灰色）
- **-**：减法（蓝色）

### 第四行
- **1 2 3**：数字按钮（浅灰色）
- **=**：等号跨两行（粉色，跨第四、五行）

### 第五行
- **0**：数字0，双倍宽度（浅灰色）
- **.**：小数点（浅灰色）
- **+**：加法（蓝色）

## 按钮样式

### 数字按钮
- 颜色：浅灰色 `#F5F5F5`
- 字体：32rpx，中等粗细
- 圆角：12rpx

### 运算符按钮
- 颜色：蓝色 `#4A90E2`
- 字体：32rpx，加粗
- 圆角：12rpx

### 清除按钮
- 颜色：红色 `#FF6B6B`
- 字体：32rpx，加粗
- 圆角：12rpx

### 等号按钮（特殊）
- 颜色：粉色 `#FF6B9D`
- 字体：44rpx，加粗
- 高度：220rpx（跨两行）
- 圆角：12rpx
- 阴影：粉色阴影效果

## 使用方法

1. **输入数字**：点击数字按钮（0-9）
2. **小数点**：点击"."按钮
3. **运算符**：点击+、-、×、÷按钮
4. **计算结果**：点击等号按钮
5. **清除**：点击C按钮

## 功能特点

- 支持四则运算（+、-、×、÷）
- 支持小数计算
- 除数为0时会提示错误
- 等号按钮跨两行设计，更易点击
- 所有按钮有点击动画反馈
- 按钮有轻微阴影效果
